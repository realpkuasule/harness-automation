# 03-generators-content.test.ts — 生成器输出内容验证

**Priority**: P1
**Files under test**: `src/generators/*.ts`

## CLAUDE.md Generator (generateClaudeMd)

### Input
```typescript
decisions: RuleDecision[]  // 包含 cognitive / claude.md / 其他 medium 的混合
```

### Test Cases (5 tests)

| # | Case | Expect |
|---|------|--------|
| 1 | 空 decisions | 输出 != "", 不含 rule 块 |
| 2 | 仅 cognitive rules | 输出含 "## 认知层规则" 和所有 rule 块 |
| 3 | 仅 soft constraints | 输出含 "## 软约束规则" |
| 4 | 混合类型 | 三个 section 都存在 |
| 5 | projectName 参数 | 标题含 projectName |

### 内容结构断言

生成的 CLAUDE.md 必须：
- 第一行是 `# <projectName> — Harness Rules`
- 含 `<!-- Auto-generated by Harness Automation System -->`
- 末尾含 `*Generated by Harness Automation System v1.0*`
- cognitive rules 含 `- **认知层辅助**: 触发 "xxx" Skill`
- reference rules 含 `→ 由 \`linter\` 自动执行`
