## G-06: export_rules response schema too vague

### Source
gap-analysis-202604280019.md §G-06

### Current state verified
```yaml
# docs/api/openapi.yaml:478-484
properties:
  export:
    type: object
  savedPath:
    type: string
    nullable: true
```

Implementation (actual structure):
```typescript
// rule_io.ts:8-25
interface RuleExportData {
  version: string;           // "1.0"
  exportedAt: string;         // ISO datetime
  source: {
    projectDir: string;
    projectPhase?: string;
    teamSize?: string;
    techStack?: string[];
  };
  rules: Array<{
    ruleId: string;
    ruleName: string;
    recommendedMedium: Medium;
    confidence: number;
    alternativeMedia: Medium[];
    reasons: string[];
  }>;
}
```

### Changes

| File | Lines | Current Content | Change | Description |
|------|-------|----------------|--------|-------------|
| `docs/api/openapi.yaml` | L478-484 | `export: type: object` (anonymous) | Replace with `$ref: "#/components/schemas/RuleExportData"` | Add structured schema |
| `docs/api/openapi.yaml` | components | New schema | Add `RuleExportData` definition | Define the structure |

New schema in components:
```yaml
RuleExportData:
  type: object
  description: 规则导出数据结构
  properties:
    version:
      type: string
      description: 导出格式版本（当前 "1.0"）
    exportedAt:
      type: string
      format: date-time
      description: 导出时间
    source:
      type: object
      description: 导出源信息
      properties:
        projectDir: { type: string }
        projectPhase: { type: string }
        teamSize: { type: string }
        techStack:
          type: array
          items: { type: string }
    rules:
      type: array
      description: 导出的规则列表
      items:
        type: object
        properties:
          ruleId: { type: string }
          ruleName: { type: string }
          recommendedMedium: { $ref: "#/components/schemas/RuleMedium" }
          confidence: { type: number }
          alternativeMedia:
            type: array
            items: { $ref: "#/components/schemas/RuleMedium" }
          reasons:
            type: array
            items: { type: string }
```

And update the response to use:
```yaml
properties:
  export:
    $ref: "#/components/schemas/RuleExportData"
  savedPath:
    type: string
    nullable: true
```

### Verification
